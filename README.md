# PokeApiAng21

Este proyecto fué desarrollado con Angular version 21.

## librerías utilizadas

Para css: bootstrap y angular material.

Propias de angular: 
- httpClient en el servicio para las peticiones manejado con observables
- RxJs para map, of, tap, etc
- Manejado con señales para inputs y cambios de estado
- Effect para reaccionar a un cambio de estado
- viewchild para referencias
- manejo de sonido básico
- y mas...
